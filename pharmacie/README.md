# VirtualRetail

Landing page générée automatiquement.

## 📁 Structure
- `index.html` - Page principale
- `test.html` - Page de test simulation
- `assets/images/` - Images à ajouter

## 🖼️ Images requises
- `hero.jpg` - Image principale (ratio 16:9)
- `solution-1.jpg` - Image solution 1 (ratio 16:9)
- `solution-2.jpg` - Image solution 2 (ratio 16:9)
- `solution-3.jpg` - Image solution 3 (ratio 16:9)

## 🚀 Déploiement
```bash
npm run deploy pharmacie
```

## 🔧 Modification
Éditer `_config/pharmacie.json` puis regénérer :
```bash
npm run build pharmacie
```
